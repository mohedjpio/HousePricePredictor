# User Guide

Welcome to **HomeVal**, the AI-powered House Price Predictor designed to help you accurately valuate properties across Egypt seamlessly.

## Getting Started

1. Double-click the desktop shortcut, or ask your system administrator to start the application for you. You will be greeted by the **Estimator** page.
2. Ensure you have network connectivity if you intend to use the AI Chat feature.

## Application Tabs

The program contains three main tabs along the top bar:

### 1. 🏠 Estimator
This is the core of HomeVal. 
- **Property Details**: Enter the property's size in square meters, location, condition, finishing status, and the number of rooms.
- **Amenities**: Toggle on extras such as security, a pool, or an elevator, as these impact the overall value.
- **Estimate Price**: Click the primary estimate button. Your inputted values are fed into our pricing model, and a realistic market price will be returned. 
- *Note*: Any estimates you make will be securely saved into a local dataset for future model improvements. 

### 2. 📊 Analytics
Provides visual graphs, charts, and breakdowns explaining housing market trends in Egypt so that you can make data-driven buying decisions. 

### 3. 💬 AI Chat
An integrated assistant ready to provide real-estate advice.
- **Groq API Key**: Need an API key? You can add it seamlessly in the top right of this section. (Or ask an admin to configure the `.env` settings).
- **Questions**: Ask about the best areas to invest in or inquire about tips when renovating a semi-finished apartment.

## Settings
- **Language (`EN`/`AR`)**: Available on the top right, this shifts the entire UI between English and Arabic.
- **Theme (`☀`/`☾`)**: Switch between Dark Mode and Light Mode.
